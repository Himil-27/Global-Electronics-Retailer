# Global-Electronics-Retailer
Sales decline and demand analysis

# Problem Statement
The business experienced strong growth until 2019, followed by a sharp decline in revenue and order volume. This project aims to identify the key drivers behind this decline and evaluate whether it is caused by demand-side or operational factors.

# Objectives
- Analyze sales trends over time
- Evaluate Average Order Value (AOV)
- Identify demand patterns and seasonality
- Assess delivery performance
- Provide actionable business recommendations

# Tools Used
- SQL
- Power BI
- Excel

# Key Insights
- Revenue decline after 2019 is primarily driven by drop in order volume
- AOV shows a gradual decline, contributing as a secondary factor
- Delivery performance remained stable, ruling out operational issues
- Strong seasonal trends, with peak sales in December

# Recommendations
- Improve customer demand through targeted strategies
- Strengthen online sales channels
- Implement upselling and cross-selling strategies
- Focus on high-value product categories
- Introduce minimum order value incentives
